# Rainfall Prediction using Machine Learning

## Project Overview

This project predicts rainfall using Machine Learning techniques on the Australian Weather Dataset.

The project includes:
- Data preprocessing
- Feature engineering
- Handling missing values
- Encoding categorical variables
- Model training
- Model evaluation
- Visualization

---

## Dataset Information

Dataset Used:
Australian Weather Dataset

Target Variable:
RainTomorrow

Dataset Source:
https://www.kaggle.com/datasets/jsphyg/weather-dataset-rattle-package
